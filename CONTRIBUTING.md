# Contributing to billariz-ui

Thank you for your interest in contributing to Billariz!

## How to contribute

1. Fork the repository
2. Create a branch: `git checkout -b feature/my-feature`
3. Commit your changes: `git commit -m 'feat: add my feature'`
4. Push to your fork: `git push origin feature/my-feature`
5. Open a Pull Request

## Code conventions

- Follow standard React/JavaScript conventions
- Run `yarn lint` before submitting
- Keep components small and focused

## Reporting bugs

Open an issue on GitHub with a clear description and steps to reproduce.
